Commands





You interact with the Service via Github issue comments, called commands.

A command will always be addressed to @charlesmike

More types of commands are understood. The engine behind is made of regular expressions. Bellow you can find a detailed table:


TypeRegExExampleEffect
Hello ^%s[ ,]+(hi|hello|how are you|how do you do|who are you|what are you|hi there|hello there)[ \\?,\\.;!]*$ @charlesmike hi there!! The bot will answer explaining what it can do for you.
Index site ^%s[ ,]+(please index|pls index|index please|index pls|index)[ \\?,\\.;!]*$ @charlesmike pls index The bot will crawl and index the whole website if you are the repo owner. See bellow a detailed explanation.
Index page ^%s[ ,]+(index \\[this\\] *\\(.+\\) page|please index \\[this\\] *\\(.+\\) page|pls index \\[this\\] *\\(.+\\) page)[ \\?,\\.;!]*$ @charlesmike pls index [this](link/to/page) page The bot will crawl and index the specified page if you are the repo owner. See bellow a detailed explanation.

In order to give certain commands you need to be the repository's owner. This means that:

  • The repo is under your name.
  • The repo is not a fork.
  • If the repo is under an organization, you must be one of the organization's active admins.
    • In this case, you need to invite @charlesmike to be a member of the organization because otherwise it won't be able to check your membership. See here.

About @charlesmike

Charles Michael is a Github chatbot which crawls and indexes your website's content making it searchable. You simply have to add a JS script to your page to render the search box.
