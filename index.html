About


Charles Michael is an easy-to-use web-content indexer. You interface with it via Github issue comments. For now, it accepts 3 types of commands: hello, index-website and index-page. Say "@charlesmike hi there!" in a comment and check out the answer!

Charles is hosted in the AWS cloud and indexes your data into ElasticSearch instances! When an index command is successfully run you will receive a JS script to put on your website (this only needs to be done the first time) and bam: the search functionality is there out of the box.

This service is available for Github-hosted websites/blogs and is completely free. In the future it might be available for other websites as well.

About @charlesmike

Charles Michael is a Github chatbot which crawls and indexes your website's content making it searchable. You simply have to add a JS script to your page to render the search box.
